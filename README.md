## `For Termux/Ssh/Ubuntu`

```bash

apt update

apt upgrade

pkg update && pkg upgrade

pkg install bash

pkg install libwebp

pkg install git -y

pkg install nodejs -y 

pkg install ffmpeg -y 

pkg install wget

pkg install yarn

pkg install imagemagick -y

git clone https://github.com/LuciZR/KHALIFA-MD

cd Maria-Md

rm -rf session

yarn install

npm start

```
